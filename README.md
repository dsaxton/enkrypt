```shell
enkrypt 0.1.0
Minimal XOR encryption utility

USAGE:
    enkrypt --key <KEY> --input <INPUT> --output <OUTPUT>

OPTIONS:
    -h, --help
            Print help information

    -i, --input <INPUT>
            Input file name

    -k, --key <KEY>
            Encryption key

    -o, --output <OUTPUT>
            Output file name

    -V, --version
            Print version information
```
